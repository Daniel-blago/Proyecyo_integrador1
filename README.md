
# Página Web Personal – Daniel Blacio

##  Descripción

Este proyecto consiste en el desarrollo de una página web personal estática utilizando HTML5 y CSS3. Su objetivo es evidenciar el uso correcto de la estructura semántica de HTML, la organización del contenido y la implementación de elementos básicos como navegación interna y formularios.

El enfoque principal está en la correcta aplicación de HTML, no en el diseño visual avanzado.

---

## Estructura del HTML

El sitio está construido utilizando etiquetas semánticas obligatorias:

* `header` → Contiene el menú de navegación
* `main` → Contenedor principal del contenido
* `section` → Organización por bloques de información
* `footer` → Información de contacto y enlaces externos

---

##  Secciones implementadas

* **Inicio:** Presentación con nombre y descripción
* **Acerca de mí:** Información personal (edad y perfil profesional)
* **Proyectos:** Áreas de desarrollo e intereses técnicos
* **Servicios:** Capacidades y soluciones ofrecidas
* **Certificaciones:** Formación y conocimientos adquiridos
* **Hobbies:** Intereses personales
* **Contacto:** Formulario para comunicación

---

##  Funcionalidades

* Navegación interna mediante enlaces (`#id`)
* Formulario de contacto con validación básica (`required`)
* Enlaces externos a redes profesionales
* Imagen de perfil cargada desde una URL externa

##  Estilos (CSS)

Se aplicaron estilos básicos cumpliendo los requisitos mínimos:

* Color de fondo de la página
* Color de texto
* Márgenes y espaciado
* Uso de `flexbox` para la barra de navegación
* Media query básica para adaptar el menú en pantallas más grandes

---

##  Tecnologías utilizadas

* HTML5
* CSS3

---

##  Estructura de archivos

* `index.html` → Estructura principal del sitio
* `styles.css` → Estilos aplicados

> Nota: No se utiliza carpeta local de imágenes, ya que la imagen de perfil se carga mediante un enlace externo.
